# AI Sentiment Analyzer

A modern **Spring Boot** web application that performs real-time sentiment analysis on any text using rule-based AI.

## Features
- Clean, responsive UI
- Real-time sentiment detection (Positive / Negative / Neutral)
- Confidence scoring
- Built with Spring Boot + Thymeleaf

## Tech Stack
- Java 21
- Spring Boot 3.3
- Maven
- Thymeleaf

## How to Run Locally

1. Clone the repository
2. Run `mvn spring-boot:run`
3. Open browser → http://localhost:8080
